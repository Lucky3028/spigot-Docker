# spigot-Docker

This repository will help you to configure Minecraft JE Spigot server on Docker.

## Description

Dockerでspigotサーバーを立ち上げるためのDockerファイル群。mysql及びphpmyadminも同時に立ち上げるようになっているので、不要ならば適宜docker-compose.ymlを編集してご使用ください。

## Requirement

* Docker
